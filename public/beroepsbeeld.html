Avanade

Bedrijfscultuur

Bij Avanade heb je veel eigen regie, en zelf discipline nodig. Managers en projectontwikkelaars zijn er om jouw te helpen, en behandelen je als gelijke.

Functies

  • IT Consultant
  • Projectontwikkelaar
  • Projectleider
  • Expert of deskundig in één of meerdere producten relevant voor het project. Bijvoorbeeld: Een Oracle expert.
  • Systeembeheerder

Technieken

  • Docker
  • Hypervisor
  • VMWare
  • MySQL
  • Redhat Linux
Meer lezen Avanade is een IT consultancy bedrijf waar ze bedrijven helpen door hen te adviseren en te begeleiden met IT oplossingen. Ik vertelde wat over mijzelf, zoals hoe bevredigend ik het vindt om anderen te kunnen helpen met een computer gerelateerde aankoop en ik heb een aantal vragen gesteld over hoe stage werkt. Ze nemen helaas alleen examen studenten aan omdat ze die over het algemeen willen aannemen, desalniettemin klinkt het wel interessant.

Ik vroeg naar de verwachtingen van de functie van zo’n pas aangenomen student. De student is nu een IT consultant die onder begeleiding van een senior consultant ervaring opdoet. De IT consultant moet een rijbewijs hebben, daar hij altijd ergens anders werkt. Je hebt als IT consultant dus geen vaste uren of een vaste werkplek, omdat je werkt bij het bedrijf dat jouw inhuurt als consultant. Op zich erg leuk, omdat ik dan altijd nieuwe plekken bezoek en mensen ontmoet, maar ik denk wel om zo snel mogelijk aan te passen aan de ongeschreven regels, ik een stuk meer ‘people skills’ moet aanleren. Of op z’n minst meer zelfvertrouwen in mijn vaardigheden.

Uiteraard, heb ik ook gevraagd naar het soort IT oplossingen/technieken die een consultant moet aanraden of kennen. Over het algemeen gaat het over server implementaties, zoals wat voor type hardware past bij het budget en wat voor type software past bij de implementatie. Dat valt precies in mijn straatje, niet zozeer omdat ik een server guru ben, (verre daarvan) maar omdat hier mijn interesse ligt en ik erg weinig weet over de server-specifieke applicatie workloads. Uiteraard als het gaat om software die veel gebruik maakt van AVX512 instructies, is Xeon de enige optie, maar de horizon is breder dan één software of merk. Uiteraard is perf/Watt het allerbelangrijkste in een server.

  • Front-end developer
  • Back-end developer

Technieken

  • Microsoft Azure
  • WebAPI
  • AngularJS
  • C#
  • CSS3
Meer lezen Betabit is een softwarebedrijf wat bedrijven op verschillende manieren helpt. Ze werken exclusief met Microsoft software en services waar mogelijk om hun klanten te helpen. Betabit geeft onder andere les aan medewerkers van hun klanten om ook developer te worden, of hun kennis te verdiepen. Beide passen wel bij mij, daar ik het leuk vindt om dingen uit te leggen en ik al ervaring heb met veel verschillende Microsoft producten, zoals Windows Server bijvoorbeeld. Anders bied Betabit op maar gemaakte cloud en software oplossingen voor hun klanten. Ik liet mijn kaartje achter op de Betabit stalling tijdens het Job event.

Gemeente Den Haag

Bedrijfscultuur

De ICT afdeling van gemeente Den Haag, heeft heel erg veel autonomie gekregen. Interessant is dan ook de dynamiek tussen het flexibele, moderne ICT team en de slome, hardnekkige hiërarchische bureaucratie.

Functies

  • Software engineer
  • IT consultant
  • Security expert

Technieken

  • Windows 7 en XP
  • Java
  • Javascript
Meer lezen De gemeente draagt de verantwoordelijkheid over vele dingen, onder andere het schoonhouden van de stad en alle administratie omtrent uitgifte van vergunningen. Om al die taken uit te kunnen voeren heeft de gemeente ook haar eigen IT team. Ik heb op het Job event met de manager van dit team gesproken in een zogenaamde bedrijven speeddate.

De gemeente staat voor unieke uitdagingen, daar innovatie en implementatie juist in die sector extra moeizaam is. Hij ging in geur en kleur beschrijven hoe hij denkt over zijn incompetente bureaucraten, zowel onder hem als boven hem. Ik beschreef hoe ik zo’n probleem zou aanpakken in de vorm van microservices, door iedere functie van de IT afdeling onafhankelijk te maken van het geheel, om zo het monolithische systeem dood te knijpen.

Hij kon mijn verhaal erg waarderen en wilde graag meer over mij weten. Werken bij de overheid zie ik wel zitten, daar er een grote uitdaging ligt in her informeren en overtuigen van onwetende mensen. Volgens hem zijn mensen met een overbruggende functie tussen het bedrijfsleven en de IT, het belangrijkst om een project tot een succes te maken. Ik heb met plezier mijn kaartje achter gelaten.

O'Moda

Bedrijfscultuur

Functies

  • Front-end developer
  • Java developer
  • Data scientist
  • Software engineer

Technieken

  • Java
  • Python
Meer lezen O’Moda is een Nederlands familiebedrijf gespecialiseerd in schoenen. Ik ben in het hoofdkantoor geweest tijdens de bedrijvensafari met mijn klasgenoten. O’Moda ontwerpt schoenen in Nederland en distribueert ze in Nederland en West-Europa. Wat dit betekent voor de IT, is dat er in het hoofdkantoor nog veel verouderde apparatuur staat, die gebruik maken van een bijna uitgestorven programmeertaal, ongeveer op het niveau van Assembly. Ze willen graag overstappen naar Java omdat Java overal kan werken en omdat er letterlijk nog maar twee mensen zijn in Europa die de oude programmeertaal kennen van hun machines.

Syntess

Bedrijfscultuur

Syntess is wat je zou kunnen beschrijven als een 'normaal' bedrijf. Er is een duidelijke hiërarchische structuur, maar het is niet zo stug en afstandelijk als de bureaucratie van de overheid.

Functies

  • Software engineer
  • Data scientist

Technieken

  • Javascript
  • MySQL
  • Java
  • Syntess Atrium (eigen software)
Meer lezen Ik ben bij Syntess geweest tijdens de bedrijvensafari. Syntess is net als Q42 een klassiek software bedrijf waarbij ze voor klanten software ontwikkelen en onderhouden. Bij Syntess werden we uitgedaagd om in teams zoveel mogelijk goed vliegende papieren vliegtuigjes te maken, mijn team was de beste. Deze uitdaging werd gebruikt als aanleiding voor het verhaal van de directeur. Hij legde uit hoe Syntess zijn eigen software pakketten aanpaste voor zijn klanten en dat Syntess al bestaat sinds 1987.

In tegenstelling tot Q42, komt Syntess 'serieuzer' over: De directeur was in pak, evenals de manager, zelfs de website is 'serieuzer', waar je bij Q42 werknemers kunt sorteren op hoe vaak ze World of Warcraft spelen, krijg je bij Syntess meteen een aanbod van verschillende software paketten.

Q42

Bedrijfscultuur

Q42 geeft de indruk van een plat bedrijf waarin er onderling geen onderscheid wordt gemaakt tussen functies. De mensen die ik ontmoette op het Job event gaven de indruk echte nerds te zijn, op een goede manier.

Functies

  • Full-stack developer

Technieken

  • HTML5
  • CSS3
  • Javascript
  • Python
  • Java
Meer lezen Toen ik bij het Job event was zag ik een stalling met wat leek op een server rack met managed switches, als afgestudeerd netwerkbeheerder kon ik dat uiteraard niet negeren.

Daar heb ik twee medewerkers ontmoet die mij van alles vertelden over hun apparaat, (het waren allerlei Raspberri Pi’s met elkaar in verbinding) en mij een programmeertest wilden laten doen. Dat moest ik helaas afslaan, daar ik nog geen Java kan, maar het was al snel heel erg duidelijk dat dit bedrijf alleen software developers en engineers zoekt. Op de banier van hun stalling stond letterlijk ‘’Full-stack developer’’ en welke programmeertalen.

Q42 is dan ook een heel duidelijk bedrijf: Ze ontwerpen en bouwen software voor de hoogste bieder. Uiteraard liet ik ook hier mijn contact kaartje achter.

YourSurprise

Bedrijfscultuur

YourSurprise is een plat bedrijf waarin er geen onderscheid in de omgang wordt gemaakt tussen functies. Er staat een glijbaan in de kantine, dat zegt genoeg.

Functies

  • Full-stack developer
  • Front-end developer (specifiek voor de personalisatie app op de site)
  • Data scientist
  • Back-end developer (specifiek voor de vertaling van de hogere programmeertalen naar lagere programmeertalen voor de machines)

Technieken

  • Javascript
  • NoSQL
  • Mobile-first
  • Machine learning
  • HTML5 en CSS3
Meer lezen Ik ontmoette YourSurprise toen ik met mijn klasgenoten naar bedrijvensafari ging. Het was de CISO die aan ons presenteerde, hij gaf ons nog een rondleiding door het nieuwe hoofdkantoor, alleen was het nieuwe hoofdkantoor een veredelde bouwput. Ik kon mij goed voorstellen hoe het er uiteindelijk uit zou zien, en ik kon me voorstellen waarom hij zo trots is op het nieuwe hoofdkantoor en waarom hij het graag wilde laten zien. Desalniettemin, YourSurprise werd 15 jaar geleden opgericht. Ze maken gepersonaliseerde cadeaus en producten voor klanten in Europa.
